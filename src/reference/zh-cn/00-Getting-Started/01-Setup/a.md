在 macOS 上安装 sbt
---------------------

### 通过通用的包安装

下载 [ZIP][ZIP] 或者 [TGZ][TGZ] 包并解压。

### 通过第三方的包安装

> **注意：** 第三方的包可能没有提供最新的版本，请记得将任何问题反馈给这些包相关的维护者。

#### 通过 [Homebrew](https://brew.sh/) 安装

```
\$ brew install sbt
```

#### 通过 [SDKMAN!](https://sdkman.io/) 安装

```
\$ sdk install sbt
```
